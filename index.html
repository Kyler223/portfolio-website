About Me

Introduction To Programming

My name is Kyler Nelson. I started to code 5+ years ago. I started out making apps on Android Studio when I was 16 years old. They weren't very good apps because it was my first time coding. But I just loved the process of writing code and seeing it appear right on my phone. That was an amazing feeling, which has never faded over the years of my coding career.

In 2020, I began to really enjoy creating Minecraft plugins. That's where I got most of my Java experience from. Then in 2021 I used ExtendScript (very similar to JavaScript) to make a script for Premiere Pro. This script automated the process of making subtitles for my videos. Then this JavaScript knowledge translated to web development.

Web Development

I have been learning front-end web development from online courses on Codecademy. I currently have 4 certificates from learning HTML , CSS , JavaScript , and React . On top of that, I have been freelancing on the side with Upwork .

The first real website I made is this Investment Calculator . Then, I'm still currently working on this Fashion Template Website . My goal is to have a remote job / freelance full time in web development in the near future.
